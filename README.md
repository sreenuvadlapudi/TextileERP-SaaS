# TextileERP-SaaS

A modern, multi-tenant ERP platform for textile manufacturers, wholesalers, and retailers.

## Overview

TextileERP-SaaS is a cloud-based ERP solution designed to simplify textile business operations, including inventory management, sales, purchases, customer management, supplier management, billing, reporting, and AI-powered business insights.

Built with the MERN stack, the platform is designed to support multiple textile businesses through a customizable Software-as-a-Service (SaaS) architecture.

## Features

- Multi-Tenant SaaS Architecture

## Installation

### Clone Repository

```bash
git clone https://github.com/sreenuvadlapudi/TextileERP-SaaS.git
```

### Install Frontend

```bash
cd client
npm install
```

### Install Backend

```bash
cd ../server
npm install
```

### Start Backend

```bash
npm run dev
```

### Start Frontend

```bash
npm run dev
```
